# Support

## Questions and discussion

Use GitHub Discussions for usage questions and general discussion.

If Discussions are not enabled on this repository, open a GitHub Issue and prefix the title with `[QUESTION]`.

## Bugs

Open a GitHub Issue and include:
- git-hex version (`cat VERSION`) and commit SHA (if pinned)
- mcp-bash version (`mcp-bash --version`)
- OS + Git version (`git --version`)
- exact tool call and arguments (redact secrets)

## Feature requests

Open a GitHub Issue using the feature request template and include example usage.

## Security issues

Do not open public issues. Follow `SECURITY.md`.
